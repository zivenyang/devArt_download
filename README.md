# devArt数据集下载器
* 本数据集出自[http://disi.unitn.it/~sartori/datasets/](http://disi.unitn.it/~sartori/datasets/)，请按照该网站声明使用
* 本数据集由500条[图片链接](http://disi.unitn.it/~sartori/datasets/deviantart-dataset/)爬取而成，其中有效下载图片为464张，另外有36条无效链接(404 Eorror)

# 使用说明
## 环境：
* Python3.x
## 主要库：
* re
* urllib
* os
## 使用方法
```
cd xxx/devArt_img
python ./download.py
```
